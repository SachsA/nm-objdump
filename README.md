# nm-objdump

### Goal
###### The project : Recode functions.
###### nm;
###### objdump;

### Program compilation:
##### Compile my_nm and my_objdump
  ```sh
        make
```
##### Compile my_nm
  ```sh
        make nm
```
##### Compile my_objdump
  ```sh
        make objdump
```

### Usage
**These two binaries work with :**
• Relocatable files .o
• Shared files .so
• Executable files

##### How to use it ?!?!
```sh
        ./my_nm [file].[extension]
```
```sh
        ./my_objdump [file].[extension]
```
